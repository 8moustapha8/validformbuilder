      Serialize, compress and encode the entire form including it's values

      serialize(boolean $blnSubmittedValues = true) : String

      Parameters

      $blnSubmittedValues

      boolean

      Whether or not to include submitted values or only serialize default values.

      Returns

      StringBase64 encoded, gzcompressed, serialized form.

      Use an array to set default values on all the forms children.

      setDefaults(array $arrDefaults = array()) 
      The array's keys should be the form name to set the default value of, the value is the actual value or values to set. Example 1 - Basic defaults: ``` addField("cool", "Coolest PHP Library", ValidForm::VFORM_STRING); //*** Set field 'cool' default value to "ValidForm Builder" $objForm->setDefaults([ "cool" => "ValidForm Builder" ]); ?> ``` Example 2 - An array of defaults: ``` addField("cool", "Cool checklist", ValidForm::VFORM_CHECK_LIST); $objCheck->addField("Option 1", "option1"); $objCheck->addField("Option 2", "option2"); $objCheck->addField("Option 3", "option3"); //*** Check options 2 and 3 by default using setDefaults() $objForm->setDefaults([ "cool" => ["option2", "option3"] ]); ?> ```

      Parameters

      $arrDefaults

      array

      The array of default values. Keys are field names, values strings or arrays
