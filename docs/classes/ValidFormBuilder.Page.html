__initializeMeta()

__initializeMeta() :

Filter out special field or label specific meta tags from the main meta array and add them to the designated meta arrays __fieldmeta or __labelmeta.

Example: $meta["labelstyle"] = "width: 20px"; will become $__fieldmeta["style"] = "width: 20px;"; Any meta key that starts with 'label' or 'field' will be assigned to it's corresponding internal meta array.
